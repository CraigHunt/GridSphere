Layout Component Events

Just like in AWT or Swing libraries, portlet layout components can receive events by implementing a listener.

The current component events are:

  • {@link org.gridlab.gridsphere.layout.event.PortletFrameEvent}
  • {@link org.gridlab.gridsphere.layout.event.PortletTabEvent}
  • {@link org.gridlab.gridsphere.layout.event.PortletTitleBarEvent}

The corresponding listeners are:

  • {@link org.gridlab.gridsphere.layout.event.PortletFrameListener}
  • {@link org.gridlab.gridsphere.layout.event.PortletTabListener}
  • {@link org.gridlab.gridsphere.layout.event.PortletTitleBarListener}
